Technology Used


The technique used in giving the solution i.e. classifying the disease in a crop uses the concept of image processing with Artificial Intelligence. More precisely, our proposed solution includes using a Deep Learning algorithm(Convolutional Neural Network) and performing other mathematical calculations on feed-forward network.


The crop recommendation tasks will be done by a Machine Learning algorithm which will calculate the similarity between the given conditions entered by the user (farmer) and pre-stored data.
